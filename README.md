# Crypto Funding Rate & Basis Monitor

Python-based monitoring script that retrieves perpetual futures funding rates and mark prices from Binance and compares them with spot prices.

## Features

- Real-time funding rate tracking
- Spot vs futures basis calculation
- Anomaly alerts for extreme funding / basis deviations
- Designed to mimic trading operations monitoring workflows

## Stack

- Python
- Requests
- Binance API
